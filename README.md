I took this project as part of my work on a BCI system complemented with an eye tracker. The keyboard module was adjusted to solve permission-related issues by using sudo and modifying specific sections of the code.

Part of the code was refactored using pynput, indentation issues were fixed, and a MediaPipe-compatible library was used to ensure better integration and stability.
At this stage, further improvements are still needed to achieve higher precision in the facial recognition component.
